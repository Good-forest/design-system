10 — Glassmorphism

Cards sur fond photo

Le glassmorphism s'applique uniquement quand une card est posée sur une image de fond (hero banner, carte de visualisation satellite). Sur fond uni, on utilise le style solid standard (.glass-panel).

✅Glassmorphism — quand l'utiliser

  • ✓Card posée sur une photo (hero banner forêt)
  • ✓Card posée sur la carte satellite interactive
  • ✓Overlay de popup sur visualisation
  • ✓Tooltip flottant sur image

❌Solid panel — à utiliser sur fond uni

  • ✗KPI cards sur fond app (#F5F7FA)
  • ✗Cards dans les listes et tableaux
  • ✗Panels dans le sidebar
  • ✗Modales sur fond opaque

11 — Map Pins

Marqueurs carte · Sévérité & Clusters

Pins SVG personnalisés pour la carte satellite. Chaque pin encode la sévérité par couleur et porte une silhouette d'arbre goodforest. Disponibles en 3 tailles et en version cluster.

Principe d'encodage

  • Rouge — Alertes (forte sévérité)
  • Orange — Anomalie (sévérité modérée)
  • Amber — Stress (surveillance)
  • Vert — Normal / sain

Implémentation

  • → Leaflet : L.divIcon avec SVG inline
  • → MapLibre / Mapbox : image source PNG ou SVG via addImage
  • → Taille standard : 32×40 px · anchor point : bas centre
  • → Taille cluster : 44×44 px · circulaire
